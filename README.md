# Stock Portfolio Tracker

A WPF desktop application for tracking stock portfolios using real-time data from the Finnhub API.

## Features

- Add stock positions (ticker, quantity, purchase price)
- Fetch current stock prices from Finnhub
- Calculate profit and loss
- Display total portfolio value

## Technologies

- C#
- .NET
- WPF
- REST API (Finnhub)
